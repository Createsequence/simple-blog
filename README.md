## 概述

简单博客2.0，基于简单博客的基础上进行重构与扩展的一套前后端分离的博客系统。

前端包括管理前台与展示前台，基于 Vue 与 ant-design-vue。

后端为基于 SpringBoot + Mysql  的 Restful 接口服务器。使用 Shrio 作为权限管理工具，并且使用 Redis 实现缓存。
